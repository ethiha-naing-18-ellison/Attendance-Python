Welcome! This is a brand new standalone project for generating simple Excel
files with raw punch data.

┌────────────────────────────────────────────────────────────────────────────┐
│ 🎯 WHAT THIS PROJECT DOES                                                  │
└────────────────────────────────────────────────────────────────────────────┘

Generates Excel files with ONE sheet called "Data" containing:

  ┌─────────────┬───────────┬──────┬──────┬──────┬──────┬──────┬──────┐
  │ Employee ID │ Name      │  In  │ Out  │  In  │ Out  │  In  │ Out  │
  ├─────────────┼───────────┼──────┼──────┼──────┼──────┼──────┼──────┤
  │    101      │ John Doe  │ 8:00 │12:00 │13:00 │17:00 │      │      │
  └─────────────┴───────────┴──────┴──────┴──────┴──────┴──────┴──────┘

• NO calculations (no OT, no Late, no penalties)
• Just RAW punch times from the database
• Simple and clean format
• Professional styling

┌────────────────────────────────────────────────────────────────────────────┐
│ ⚡ QUICK START (3 STEPS!)                                                  │
└────────────────────────────────────────────────────────────────────────────┘

  1️⃣  Install dependencies:
      pip install -r requirements.txt

  2️⃣  Start the server:
      python run_data_generator.py

  3️⃣  Open in browser:
      http://localhost:8001

  That's it! Upload your ZK.db file and generate reports! 🎉

┌────────────────────────────────────────────────────────────────────────────┐
│ 💡 TIPS                                                                    │
└────────────────────────────────────────────────────────────────────────────┘

  • For single day report: Only enter start_date
  • For month report: Enter first and last day of month
  • Sunday rows are automatically highlighted in yellow
  • Time format is HH:MM (no seconds)
  • All employees in date range are included
