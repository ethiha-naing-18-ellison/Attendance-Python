✨ PROJECT OVERVIEW
───────────────────────────────────────────────────────────────────────────────
This is a STANDALONE project that generates simple Excel files with raw punch
data only. It is separate from the main attendance system but can reference the
same database files.

🎯 PURPOSE
───────────────────────────────────────────────────────────────────────────────
Generate simple Excel files showing:
  • Employee ID
  • Employee Name
  • Raw punch times (In, Out, In, Out, In, Out)
  • No calculations
  • No complex formatting
  • Single "Data" sheet only

📋 OUTPUT COLUMNS
───────────────────────────────────────────────────────────────────────────────
1. Employee ID
2. Name
3. In (1st punch)
4. Out (2nd punch)
5. In (3rd punch)
6. Out (4th punch)
7. In (5th punch)
8. Out (6th punch)

🚀 HOW TO RUN
───────────────────────────────────────────────────────────────────────────────
1. Install dependencies:
   pip install -r requirements.txt

2. Start the server:
   python run_data_generator.py

3. Open web interface:
   http://localhost:8001

4. Upload ZK.db file and generate report

🔌 API DETAILS
───────────────────────────────────────────────────────────────────────────────
• Port: 8001
• Endpoint: POST /generate-data-sheet
• Input: ZK.db file + date range
• Output: Excel file with "Data" sheet only

🔧 CUSTOMIZATION
───────────────────────────────────────────────────────────────────────────────
You can easily modify:
• Port number (change in data_generator_api.py)
• Column names (change headers list)
• Colors and styling (change PatternFill values)
• Date format (modify format_time function)
• Output filename (change in create_data_excel)

💡 WHEN TO USE THIS PROJECT
───────────────────────────────────────────────────────────────────────────────
✅ When you need simple raw punch data
✅ When you want quick exports without calculations
✅ When you need to verify actual punch times
✅ When building other tools that need punch data
✅ When you want a lightweight solution

❌ When NOT to use this project:
❌ When you need attendance calculations (use main project)
❌ When you need overtime calculations (use main project)
❌ When you need complex reports (use main project)
